Perguntas frequentes

O FlowMen é realmente 100% offline?

Sim. O app não se comunica com servidor nenhum. Todos os dados ficam armazenados apenas no seu celular. Se você apagar o app, os dados são apagados junto — não há backup automático em nuvem.

A única exceção é o download da fonte Manrope via Google Fonts na primeira vez que o app abre. Esse download é público e não envia informação sobre você. Mais detalhes na Política de Privacidade.

Como faço backup dos meus dados?

No app: Ajustes → Exportar dados. O app gera um texto longo com todos os seus registros e coloca na área de transferência (clipboard). Você pode colar esse texto em:

  • Uma nota pessoal (Apple Notes / Google Keep)
  • Um e-mail pra você mesmo
  • Um app de anotações criptografadas
  • Um arquivo .txt no celular

Pra restaurar num celular novo (ou depois de reinstalar o app): Ajustes → Importar dados e cole o texto de volta.

Guarde esse backup em local seguro — ele contém todos os seus dados.

Esqueci meu PIN. Como recupero?

O PIN é armazenado localmente, protegido por hash. Não existe servidor para resetar.

A única opção é: Ajustes → Zona de perigo → Apagar todos os dados. Isso apaga o PIN junto com todos os seus registros, voltando o app ao estado inicial.

Perda total é intencional — app 100% local não tem "esqueci minha senha".

A biometria (Face ID / Touch ID) não está funcionando

  1. Confira se o celular tem biometria cadastrada no próprio sistema (Ajustes do iOS/Android → Face ID / Touch ID / Impressão Digital).
  2. No FlowMen, vá em Ajustes → Bloqueio do app e ative a biometria.
  3. Se continuar falhando, você sempre pode cair no PIN.

Como ativo o modo discreto?

Ajustes → Modo discreto → Trocar ícone e nome. Efeito:

  • Android: o ícone vira um calendário e o nome aparece como "Calendário" na tela inicial.
  • iOS: só o ícone muda (vira um calendário). O nome continua FlowMen — limitação da Apple, não dá pra mudar o nome em runtime.

A mudança pode levar alguns minutos pra aparecer no launcher — depende do celular. Se não aparecer rápido, reinicie o aparelho.

Posso registrar mais de uma parceira?

Sim. Ajustes → Parceiras → Adicionar. Cada parceira tem histórico separado. Você seleciona qual é a "ativa" a qualquer momento — é ela que aparece no Hoje, Calendário, Ciclo e Estatísticas.

Arquivar uma parceira preserva o histórico dela; reativar depois é só um toque.

Como funciona a previsão de ciclo?

A partir dos dados que você cadastrar em Ajustes → Parceiras → Editar → Ciclo menstrual (data da última menstruação, duração média do ciclo, duração da menstruação), o app calcula localmente:

  • Fase atual (menstrual / folicular / ovulatória / lútea)
  • Janela fértil
  • Previsão da próxima menstruação

Nota: isso é um estimador simples baseado em calendário. Não é método contraceptivo e não substitui orientação médica. Para contracepção, consulte um profissional de saúde.

O app me pede notificação pra quê?

Exclusivamente pra lembretes locais agendados pelo próprio app:

  • Lembrete diário de registro (horário configurável em Ajustes)
  • Alertas de datas especiais que você cadastrou (aniversário, data do casal, etc.)

Nenhuma notificação vem de servidor externo.

Meus dados são compartilhados com alguém?

Não. Com ninguém. Nem com a gente. Nem com a Apple. Nem com o Google. Nem com empresas de analytics. Nem pra fins de pesquisa agregada.

O app não tem tecnicamente como compartilhar — não há servidor pra enviar nada.

Como apago o app e todos os dados?

Duas formas:

  1. Apagar só os dados (mantém o app instalado): Ajustes → Zona de perigo → Apagar todos os dados
  2. Apagar o app inteiro: segure o ícone na tela inicial → Remover App → Excluir Aplicativo (iOS) / Desinstalar (Android). Todos os dados são apagados junto.

Tem versão para Android / iOS?

Sim, as duas plataformas. A experiência é praticamente idêntica.
